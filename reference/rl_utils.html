2.5.1. Reportlab Wrappers (layout.rl_utils)¶

This module provides helper functions to render managers to one or more pages of a reportlab canvas. To create multi-page documents, use the layout.pages.output.PagesLM class to wrap a series of single-page managers.

layout.rl_utils.render_to_reportlab_canvas(rl_canvas, papersize_tuple, manager)¶

Renders the given manager full size on a page of the given canvas.

layout.rl_utils.render_to_reportlab_document(output_filename, papersize_tuple, manager)¶

Creates a complete document in which to render the given manager.
